Frequently Asked Questions.

We got you the answer.

What is UnifiedDashboard?

Good question. UnifiedDashboard is a project started by Dimensional Brotherhood as part of their Open Source Software Libraries (as known as Dimensional OSSL)

Why bother create UnifiedDashboard in the first place?

UnifiedDashboard used to be Dimensional Internal Corporate Tools, created in 2009 for project management as well for onboarding. It's been in public beta in 2012 and it's released to the public in 2013.

What is Mainnets?

Mainnets is essentially public servers. UnifiedDashboard can be used offline (with Self-Hosted Solution) and online (with Mainnets).

Where to find Mainnets?

You can find it in dmsn.id/mainnets. Although there's some Mainnets that aren't registred with Dimensional. These are unmoderated and as such, we can't verify the content inside.

Good, how much I have to pay (for personal and corporate use)?

Zero, Tidak Ada, Nada. There's no need to pay, even to use corporate features.

Do you guys support TFA?

UnifiedDashboard can be locked using FIDO2 Keys and Passkeys. The server side can also block unregistred connections (for example, those who don't use Corporate VPN.)

Is UnifiedDashboard developer friendly?

Yes we do! All UnifiedDashboard base code are fully documented. This also includes their server, frontend, API and more!
